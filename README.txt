BESONDERHEITEN DES PROJEKTS:
Es handelt sich um eine verteilte Anwendung, bei welcher ein VR-Nutzer mit der Hilfe einer Begleitperson implementierte Szenarien
bewältigen soll. Die Szenarien können von der Begleitperson live beeinflusst werden.
Das Projekt besteht aus zwei einzelnen Apps:
Das VR-Programm muss auf der VR-Brille (Oculus/Meta Quest 2) installiert worden sein (z.B. via Meta Quest Developer Hub).
Die PC-Anwendung muss auf einem Computer mit Windows-, Mac- oder Linux-Betriebssystem installiert worden sein.
Die Oculus-App muss auf dem selben Computer installiert und erfolgreich mit der Oculus/Meta Quest 2 verbunden worden sein. 
Daraufhin kann die Anwendung gestartet werden.
Um als Begleitperson am PC die Sicht des VR-Nutzers live nachverfolgen zu können, wird das Mirroring der Oculus-App verwendet.
